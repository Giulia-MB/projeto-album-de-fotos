# 🖼️ Álbum de Fotos

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

Esse é um projeto de front-end feito para criar um álbum de fotos responsivo usando HTML e CSS. A ideia foi organizar várias imagens em colunas que se adaptam ao tamanho da tela.

## ✨ Sobre o projeto

A página apresenta um álbum com várias fotos distribuídas em colunas. Em telas menores, as imagens aparecem em uma única coluna; em tablets, passam a ocupar duas colunas; e em telas maiores, o layout se organiza em quatro colunas.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries
- Layout responsivo
- Imagens locais

## 📚 O que eu pratiquei

- Organização de imagens em uma galeria
- Criação de layout com Flexbox
- Uso de `flex-flow` para controlar direção e quebra de linha
- Ajuste de colunas com `flex`
- Responsividade com `@media`
- Separação de estilos em arquivos CSS diferentes
- Estruturação da página com `header`, `main`, `section` e `footer`

## 💡 Tipo de projeto

Esse é um projeto de estudo focado em responsividade e organização visual. Ele ajudou a praticar como montar uma galeria simples de imagens que se adapta melhor a diferentes tamanhos de tela.
